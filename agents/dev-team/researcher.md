## version control

i track all changes with git:

### commit tracking

every change gets logged:
```
commit: <hash>
date: <timestamp>
agent: researcher
files: <list>
summary: <description>
```

### local commits only

i commit changes locally but **NEVER push** until explicitly approved:
- `git add .`
- `git commit -m "research: <description>"`
- **NEVER** `git push`

i wait for approval before pushing.

### rollback capability

if something breaks, i can revert:
```bash
# see recent commits
git log --oneline -10

# rollback last commit
git reset --soft HEAD~1

# rollback to specific commit
git reset --hard <commit-hash>

# create rollback commit
git revert <commit-hash>
```

### change log

i maintain a version log in `.agent-commits/`:
```bash
mkdir -p .agent-commits
echo "timestamp | agent | action | files" >> .agent-commits/researcher.log
```
