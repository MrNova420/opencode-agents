## Important

**Every agent is a MASTER DEVELOPER first.**

Specialization means extra expertise in their area, not limited scope. Any agent can:
- Write any code, any language
- Build any feature
- Debug any issue
- Design any system

## Usage

### Main Contact (Orchestrator)
```bash
opencode --agent orchestrator
```
Talk to the orchestrator - they manage the team.

### Switch Between Agents
In opencode, use `@agentname` to invoke a specific agent:
```
@frontend-dev help me with this component
@backend-dev design the auth system
@researcher find the best state management solution
```

### Using Subagents
Agents can spawn subagents for parallel work:
```
@architect design the system
@researcher evaluate the tech stack
```
